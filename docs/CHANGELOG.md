# 📋 auto-gao 更版摘要

> 本文件記錄 **auto-gao** 各版本的功能異動與改善內容，以使用者視角呈現。

---

## 🚀 v1.2.0（2026-06-22）

### 1. 🧪 補品過濾升級

- HP/SP 補品選單現在**只顯示真正的回復道具**
- 以下類型已自動排除，不再出現在選單中：
  - 卷軸（護身卷軸、平衡卷軸等）
  - 重置券（能力重置券、天賦重置券）
  - 技能書
- 改用遊戲 API 的 `effect.type` 精準分類，**未來新增道具自動適用**，無須手動維護清單

---

### 2. 🍄 新增菇菇仙境 12F 秘徑支援

- 地圖選擇中新增「**菇菇仙境**」選項
- 到達蘑菇園 12F 後，自動進入菇菇仙境秘徑

---

### 3. ⚔️ Boss 獨奏模式升級

- 原本的「挑戰 Boss」開關升級為**三段式選擇**：

| 模式 | 行為 |
|------|------|
| 🔕 不參與（預設） | 不打 Boss，繼續正常巡邏 |
| ⏸️ 等待 | 遇到 Boss 時停下等待隊友 |
| 🏃 跳過 | 遇到 Boss 時自動逃跑 |

- **修正**：修復了當玩家在「路過打王」或「原地等王」模式下，打完 Boss 會強制被傳送回村的 Bug。現在打完 Boss 後會留在原地，繼續進行趕路爬樓或就地等待，避免中斷爬樓進度。

---

### 4. 🏷️ 標籤顯示修正

- 修正**未進入秘徑時仍顯示秘徑標籤**的問題
- 統一名稱：「秘境」→「**秘徑**」

---

### 5. 🖥️ 介面佈局改善

- 修正地圖／秘徑標籤文字被擠壓截斷的問題
- 標籤文字過長時自動換行顯示，不再溢出裁切

---

### 6. ✏️ 名稱正確化

- 全站名稱統一更正為 **auto-gao**（原 `auto-sgo` 為錯誤拼寫）
- 備份檔案名稱同步更新：`gao_backup_YYYY-MM-DD.json`

---

### 7. 📯 戰鬥日誌淨化

- 移除「戰鬥日誌」中顯示掉落物的詳細資訊，避免冗長字串洗版，提升日誌可讀性

---

## 🏗️ v1.1.0-Modern（舊版）

> 本版本為重大架構重構，升級至現代技術棧。

- ⚙️ 重構為 **Vue 3 + TypeScript + Element Plus** 現代技術棧
- 🗂️ 引入 **Pinia** 全局狀態管理
- 👥 支援**多帳號同時管理**
- 🔨 新增**鍛造收藏組合**功能
- 💾 支援**配置備份與還原**（JSON 匯出 / 匯入）
- ⛏️ 新增**自動採礦**、**自動回收**功能
- 📱 側邊欄收合 / 展開、手機版 **RWD 響應式介面**
