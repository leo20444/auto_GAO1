# 🎮 GAO 官方遊戲更新歷程

本文件記錄 GAO 遊戲官方的版本更新內容，方便對照 Bot 的功能優化與更新時機。

---

## v2.3 系列（鍛造系統 V2 與裝備元素擴充）

### v2.3.5（2026-06-23）
* **修復**：修正秘境頭目成就（綠水靈、巴洛古、殭屍菇菇的「討伐團」與「之王／單吃」）擊敗頭目後無法解鎖的問題；過往已達成的玩家已自動補發，可至成就頁領取獎勵。

### v2.3.4（2026-06-23）
* **新功能**：新增金幣上限機制：金幣上限隨等級提升，導覽列金幣顯示「目前 / 上限」，達上限時轉為金色標示。

### v2.3.3（2026-06-23）
* **新功能**：蘑菇園秘境新增 3 個成就：殭屍菇菇討伐團（擊敗殭屍菇菇王）、殭屍道長（單吃殭屍菇菇王）、弑仙（同時擊敗麻姑及何仙姑）。

### v2.3.2（2026-06-22）
* **調整**：鍛造成品的重量縮減效果在角色數值較高時改為更平緩，避免重量被壓到幾乎消失。

### v2.3.1（2026-06-22）
* **調整**：裝備負重對速度的影響加重，但力量配點可抵銷負重造成的速度懲罰。

### v2.3.0（2026-06-22）
* **新功能**：鍛造系統大幅擴充：菇傘、尾巴、靈珠、白髮白骨、滑黏液、道符、仙桃、睡蓮等均可帶入鍛造；寶石類素材（綠寶石、藍寶石、紅寶石、紫水晶、鑽石）放入鍛造可生成風水裝備。
